# Unloader Project

A Python-based GUI application (PySide6) for roller image processing and distance validation.

---

## Requirements
- Python 3.10 or higher
- pip (Python package manager)

---


## Project Enviornment Setup (Rasberry PI)
```bash
pip install uv
- After installing uv,
uv sync
```

## To Run the project
### Main CLI of the app
```bash
uv python -m main
```
### GUI Debugger
```bash
uv python -m ui.main
```
